Creating Your Account

ProspecTor requires account creation through the admin configuration. Follow these steps to create a new user:

  1. 1 Add your user details to the /config/users.yaml file
  2. 2 Run make enter-scheduler to access the scheduler container
  3. 3 Execute go run cmd/scheduler/main.go sync-users /app/config/users.yaml to create the user
  4. 4 Return to this page and sign in with your new credentials
